starting skylark.js

Skyalrk.js is a modular javascript library and application framework, uses the AMD specification, could run with the RequireJS library.

folder structure

Determine the forder structure for web site or web application. for example: 

|-index.html 
`-js/ 
  |-main.js 
  `-lib/ 
     `-skylark/... 
     `-oth-amd-lib/... 
     |-non-amd-lib.js

use requirejs

In index.html,you can use the RequireJS library downloaded , or use a RequireJS CDN Combination. 

<!DOCTYPE html> 
<html> 
<head> 
<script type="text/javascript" data-main="./main.js"  
        src="https://cdnjs.cloudflare.com/ajax/libs/require.js/2.3.3/require.min.js"> </script> 

</head> 

<body> 
</body> 

</html>

base config

Use requirejs.config () in main.js to define skylark packages and so on. 

    require.config({ 
        baseUrl: "./", 
        packages: [{ 
            name: "skylark", 
            location: "./lib/skylark" 
        }, ] 
    });

Integrate other libraries

amd libray

You can use an other AMD library, like using skylark.

non amd libray

You can use a non-AMD library, such as underscore, which follows the requirejs syntax 

require.config({ 
  shim: { 
    underscore: { 
      exports: '_' 
    } 
  } 
  , paths: { 
    underscore : "http://cdnjs.cloudflare.com/ajax/libs/underscore.js/1.5.2/underscore-min" 
  }   
   
});

replace jquery

Jquery is a most popular javascript library, with a large number of plug-ins and application using jquery.

The optional skylark.jquery library is based on the skylark/query module, providing APIs that are fully compatible with jquery,and the codes is simpler and more productive, JQuery plugins and applications can run directly on skylark without jquery.

examples: please refer to the skalark.jquery project.
