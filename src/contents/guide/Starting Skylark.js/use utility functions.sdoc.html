use utility functions

The skylark javascript library provides a large number of simple but powerful utility functions, and you can use this modules separately as needed.

Composition

The skylark javascript library provides the following utility modules：

  1. skylark/async
    This module defines some APIs about Asynchronous function
  2. skylark/browser
    This module defines some APIs about brower compatibility.
  3. skylark/css
    This module defines some APIs about stylesheet and rule.
  4. skylark/datax
    This module defines some APIs wrapping DOM attribute and property.
  5. skylark/dnd
    This module defines some APIs wrapping DOM drag and drop.
  6. skylark/eventer This module defines some APIs wrapping DOM event.
  7. skylark/filer
    This module defines some APIs wrapping local file reand and write.
  8. skylark/finder
    This module defines some APIs wrapping dom query.
  9. skylark/fx
    This module defines some APIs wrapping DOM transition and animation.
  10. skylark/geom
    This module defines some APIs wrapping DOM geometry.
  11. skylark/langx
    This module defines some APIs that extend language functionality
  12. skylark/mover
    This module defines some APIs wrapping DOM move.
  13. skylark/noder
    This module defines some APIs wrapping DOM construction.
  14. skylark/scripter
    This module defines some APIs wrapping script load and unload.
  15. skylark/styler
    This module defines some APIs wrapping dom style and class.

Usage

Refence a module object, take skylark/geom、skylark/finder and skylark/styler an example. 

define([ 
    "skylark/geom", 
    "skylark/finder", 
    "skylark/styler" 
],function(geom,finder,styler){ 
}); 
OR 
requre([ 
    "skylark/geom", 
    "skylark/finder", 
    "skylark/styler" 
],function(geom,finder,styler) { 
});

use function, to get the position of the element on the page as an example. 

  var el = finder.find(".target:first"); 
  if (el) { 
    var p = geom.pagePosition(el); 
    p.left = p.left + 10; 
    p.top = p.top + 30; 
    styler.css(el,{ 
      position : "absoute" 
    }); 
    geom.pagePosition(el,p);   
  }
